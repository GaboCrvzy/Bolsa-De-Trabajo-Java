Bolsa de Trabajos - Proyecto

Instrucciones de instalación/ejecución
1. Asegúrese de tener instalado Java JDK (versión 8 o superior).
2. Abra NetBeans o Eclipse.
3. Importe el proyecto desde la carpeta "ProyectoPOO".
4. Ejecute la clase principal `ProyectoPOO.java`.

Ejecución desde consola (opcional)
1. Navegar a la carpeta src del proyecto "ProyectoPOO".
2. Compilar: javac ProyectoPOO.java
3. Ejecutar: java ProyectoPOO
